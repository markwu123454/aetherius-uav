# aetherius-uav

Start Backend:  
<code>cd backend</code>  
<code>uvicorn main:app --host 0.0.0.0 --port 8000 --reload</code>  
  
Start Frontend:  
<code>deactivate</code>  
<code>cd frontend</code>  
<code>npm run dev -- --host</code>  
  
  
Terminate:  
<code>taskkill /F /IM node.exe</code>  
<code>taskkill /F /IM python.exe</code>  

other misc:  
<code>tasklist</code>
